### Quick Start

```bash

# Get recommendations
```

### Prerequisites
- Flutter 3.24.5+
- Python 3.10+
- Node.js 18+ (for SkillKit)
- Supabase Account
- Gemini API Key

### 1. Clone & Install


### 2. Backend Setup

