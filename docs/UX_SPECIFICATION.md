# Fitola – Product Design & Wireframes

## Overview
Fitola — A wellness, personalized fitness, and social engagement app with chat, live location sharing, AI plans, and rich privacy controls.

## Wireframes
### 1. Onboarding Flow
- Logo placement
- Google/Email Sign-in logic
- Personal details (Age, Weight, Height, Body Type, City, Goal)

### 2. Chat & Translation
- Status Translate Wheel (FAB)
- Ghost Mode toggle

### 3. Map & FitBuddy Locator
- OpenStreetMap integration
- Radius filters (5km default)
- Color-coded gender pins

### 4. AI Plans
- Weekly calorie targets
- Workout schedules (Mon, Wed, Fri)

## Navigation
START -> Onboarding -> Dashboard -> Chat/Map/AI Plans -> Referrals -> Settings
