## Key Components

### MCP Configuration (mcp.json)
```json
{
  "servers": {
    "sequential-thinking": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-sequential-thinking"]
    },
    "stitch": {
      "command": "npx",
      "args": ["@_davideast/stitch-mcp", "proxy"],
      "env": {
        "STITCH_PROJECT_ID": "YOUR_PROJECT_ID_HERE"
      }
    }
  }
}
```

### Deployment Script (deploy.sh)
```bash
#!/bin/bash
# One-command deployment
./deploy.sh production true

# Automatically:
# 1. Runs tests
# 2. Builds backend
# 3. Deploys to Vercel
# 4. Builds mobile apps
# 5. Reports status
```

### CI Workflow (.github/workflows/ci.yml)
```yaml
name: CI - Lint & Test
on: [push, pull_request]

jobs:
  # Backend tests
  # Flutter tests
  # Build validation
  # MCP validation
```

### Deploy Workflow (.github/workflows/deploy.yml)
```yaml
name: Deploy to Production
on:
  push:
    branches: [main]

jobs:
  # Deploy backend
  # Build mobile
  # Deploy web
```
